# 🏥 SaludYa - Backend (Fisioterapia)

API REST para la gestión de citas en consultorios de fisioterapia y rehabilitación.

## 🚀 Descripción

Este backend soporta la lógica de negocio del sistema SaludYa, permitiendo la gestión de servicios terapéuticos y citas.

El sistema administra:

- Pacientes
- Profesionales (fisioterapeutas)
- Especialidades
- Servicios de fisioterapia
- Horarios
- Citas

---

## 🔐 Funcionalidades

- Autenticación y autorización
- Gestión de profesionales de salud
- Gestión de servicios terapéuticos
- Clasificación de servicios por categoría:
  - Evaluación inicial
  - Terapias físicas
  - Terapias con equipos
  - Rehabilitación específica
- Asignación de servicios a profesionales
- Gestión de horarios
- Agendamiento de citas

---

## 🎯 Objetivo

Construir una API robusta que permita gestionar servicios de fisioterapia, optimizando la asignación de citas y recursos.
